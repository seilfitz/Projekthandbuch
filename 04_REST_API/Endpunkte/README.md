# Endpunkte

Dieser Ordner enthält die Detailbeschreibung der REST-Endpunkte nach fachlichen Domänen.

## Grundsatz

Die Endpunkte orientieren sich an fachlichen Anwendungsfällen, nicht an Oracle-Tabellen.

## Abgrenzung

Die Detailseiten beschreiben zunächst nur fachliche REST-Schnittstellen.

Technische Details wie konkrete Oracle-Tabellen, Packages und Datenbankzugriffe werden im Kapitel 05_Datenmodell dokumentiert.
